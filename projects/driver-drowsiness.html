Driver Drowsiness Detection

Python • OpenCV • Arduino • Computer Vision

Overview

Built a real-time fatigue detection system that uses computer vision and facial landmark analysis to detect driver drowsiness. The system monitors eye aspect ratio (EAR) and mouth aspect ratio (MAR) to determine fatigue levels and triggers alerts through an Arduino-controlled hardware interface.

How It Works

A camera captures the driver's face in real-time. OpenCV with dlib's 68-point facial landmark detector extracts key facial features. The system continuously computes:

  • Eye Aspect Ratio (EAR) – detects prolonged eye closure
  • Mouth Aspect Ratio (MAR) – detects yawning
  • Head pose estimation – detects head drooping
  • Temporal analysis – distinguishes blinks from drowsiness

Hardware Integration

When drowsiness is detected beyond a configurable threshold, the Python application sends serial commands to an Arduino board which activates a buzzer alarm and can interface with vehicle safety systems (e.g., seat vibration, dashboard warning).

Results

The system achieves reliable detection with minimal false positives under varying lighting conditions. Frame processing runs at 30+ FPS on standard hardware, making it practical for real-world deployment.
