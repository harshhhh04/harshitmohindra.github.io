Prosthetic Hand CNN

ESP32 • CNN • Embedded C • Onboard ML

Hero image – add to project-assets/prosthetic/

Overview

Designed and deployed an embedded ML inference system on ESP32 for real-time gesture classification in prosthetic hand control. The system runs a lightweight CNN model directly on the microcontroller, enabling low-latency onboard inference without cloud connectivity.

System Design

The pipeline takes sensor data as input, preprocesses it on-chip, and feeds it through a quantized convolutional neural network optimized for the ESP32's limited memory and compute resources.

  • Data acquisition from EMG/IMU sensors
  • Signal preprocessing and feature extraction
  • Quantized CNN model for gesture classification
  • Motor control output for prosthetic actuation

ML Pipeline

The CNN was trained in Python using standard deep learning frameworks, then quantized to INT8 precision and converted to a C array for deployment on ESP32. Model accuracy was validated both off-device and on-device to ensure consistent performance.

Challenges

Fitting a meaningful neural network into ESP32's constrained SRAM required careful model architecture choices — depthwise separable convolutions, aggressive quantization, and layer fusion to reduce memory footprint while maintaining classification accuracy.
