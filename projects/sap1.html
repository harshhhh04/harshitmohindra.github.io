SAP-1 Computer

Digital Design • Assembly • Architecture

Hero image – add to project-assets/sap1/

Overview

Built a Simple-As-Possible (SAP-1) educational CPU architecture that implements fundamental arithmetic and assembly operations. This project demonstrates core computer architecture concepts including instruction fetch-decode-execute cycles, bus architecture, and register-based data movement.

Architecture

The SAP-1 architecture consists of essential digital building blocks wired together through a shared W-bus:

  • Program Counter (PC)
  • Memory Address Register (MAR) and RAM
  • Instruction Register (IR) and Controller/Sequencer
  • Accumulator (Register A) and Register B
  • Adder/Subtractor and Output Register

Instruction Set

The processor supports a minimal but complete instruction set including LDA (Load), ADD, SUB, OUT (Output), and HLT (Halt). Each instruction is 8 bits wide — 4-bit opcode and 4-bit operand — allowing up to 16 memory addresses.

Key Takeaways

This project deepened understanding of how hardware executes software at the lowest level — from clock cycles driving state machines to how control signals orchestrate data flow across registers and the ALU.
